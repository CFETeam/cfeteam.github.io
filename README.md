CFETeam 团队博客
================
腾讯云前端团队博客，基于 [hexo](https://hexo.io/zh-cn/) 构建

使用
----

1.	下载源码:
  ```bash
  git clone https://github.com/CFETeam/cfeteam.github.io.git
  ```

2.	安装依赖
  ```bash
  cd cfeteam.github.io && npm install
  ```

3.  编写文章:
  ```bash
  hexo new {{title}}
  ```

4. 启动本地预览
  ```bash
  hexo server
  ```

5.	生成网站:
  ```bash
  hexo generate
  ```

6.  发布:
  ```bash
  hexo deploy
  ```
